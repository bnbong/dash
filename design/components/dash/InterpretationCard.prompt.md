**InterpretationCard** — the single most important Dash moment: the AI translating a day's actions into operating meaning, set in the calm **Pretendard (gothic) voice** — regular weight, relaxed line-height. Calm, specific, never over-praising.

```jsx
<InterpretationCard seed={1}>
  오늘은 공간과 생활 기반을 회복한 날입니다. 방 청소와 공과금 납부는 당장
  눈에 띄는 성취는 아니지만, 앞으로의 생활 마찰을 줄이는 운영 행동입니다.
</InterpretationCard>
```

Tones: `tint` (pine) · `paper` · `inverse`. Set `seed` to append the Grove reflection line.
